选择科室查看专项规则
骨科DRG核心提示：主要诊断决定进入I系列（肌肉骨骼系统）还是B系列（神经系统），影响权重差异可达数倍。

主诊选择关键规则

多部位损伤 → 选危害最重部位
如骨折合并神经损伤，应以神经损伤为主诊（B系列权重通常 > I系列），但需确有神经科诊疗证据。
脊柱手术 → 区分颈/胸/腰段
颈椎前路/后路手术分组不同，编码需精确到手术入路，避免笼统编"脊柱手术"导致低权重组。
关节置换 → 明确单/双侧、初次/翻修
髋膝关节翻修（revision）权重显著高于初次置换，必须如实标注；双侧置换同次住院应标注双侧。
合并症加权：骨科高权重MCC/CC
术后深静脉血栓（DVT）、肺栓塞、伤口感染、脂肪栓塞、ARDS等属MCC/CC，需如实记录并有治疗措施。

常见骨科高权重病组示例

脊柱融合术（多节段）
ADRG: IB1x，伴MCC权重 > 5.0，需记录节段数及内固定情况。
髋关节翻修
翻修手术权重约为初次置换1.5~2倍，须有明确翻修指征记录。
复杂骨盆骨折
合并血管损伤或腹腔脏器损伤时，主诊可考虑血管损伤（T系列），需多科会诊记录。
合规警示：骨科是医保稽核重点科室。不得仅凭骨折加上神经损伤编码而无神经科相关检查或治疗记录，否则属高套违规。
肿瘤科DRG核心提示：住院目的不同，主诊断选择规则完全不同——手术、放疗、化疗、支持治疗各有专属原则。

肿瘤主诊断选择四大场景

场景1 手术切除为主要目的
主诊选恶性肿瘤本身（如胃恶性肿瘤 C16），并记录具体手术方式（根治/姑息切除）。
场景2 单纯化疗/靶向治疗
主诊选"恶性肿瘤化学治疗"（Z51.1），原发肿瘤作为其他诊断。
场景3 单纯放疗
主诊选"恶性肿瘤放射治疗"（Z51.0），与化疗同次住院时以消耗资源多者为主诊。
场景4 并发症/急症处理
因并发症（如肠梗阻、上消化道出血）入院，主诊选并发症；原发肿瘤为其他诊断。可显著提升权重。

肿瘤科高权重MCC/CC

恶性胸腔积液、恶性腹腔积液（须有穿刺引流记录）
化疗后Ⅳ度骨髓抑制（须有血常规结果 + 治疗记录）
肿瘤溶解综合征、弥漫性血管内凝血（DIC）
脓毒血症/感染性休克（须有培养结果或影像支持）
合规警示：不得在所有化疗患者中统一加注"恶心呕吐"诊断。须有相应治疗记录方可作为CC填报。
心内科DRG核心提示：介入手术类型（PCI、起搏器植入、射频消融等）直接决定ADRG分组，比内科诊断权重高2~5倍。

心内科主诊选择规则

急性心肌梗死 → 明确部位与类型
前壁/下壁/侧壁STEMI、NSTEMI编码不同，影响ADRG分配。有PCI手术时主诊仍选AMI，手术操作填PCI。
心衰 → 明确NYHA分级 + 类型
收缩性（HFrEF）/舒张性（HFpEF）心衰编码不同；NYHA Ⅲ-Ⅳ级属MCC。
介入手术类型须精确
单支/多支血管PCI、DES/BMS支架、CTO病变PCI等细分，多支血管病变权重更高，须有影像报告支持。

心内科高权重MCC/CC

心源性休克（须有血压/血流动力学记录 + 升压药使用）
完全性房室传导阻滞（须有心电图记录）
急性肾衰竭（Cr升高 + 少尿，须记录透析或利尿治疗）
恶性心律失常：室颤/室速需电复律（须有除颤记录）
合规警示：心衰NYHA分级须与病程记录一致；PCI支架数量须与手术记录及耗材使用台账一致。
心外科DRG核心提示：手术类型和是否体外循环是分组关键，搭桥血管数目、瓣膜置换/成形方式须精确记录。

心外科主诊及手术规则

CABG → 明确体外循环/非体外循环、桥血管数目
体外循环CABG权重高于非体外，多支搭桥权重更高；搭桥数目须与手术记录一致。
瓣膜手术 → 区分置换/成形、单瓣/多瓣、生物瓣/机械瓣
双瓣置换权重远高于单瓣，合并CABG的联合手术属最高权重组，须有超声心动图支持。
主动脉手术 → 区分根部/升主动脉/弓部，全弓替换+支架象鼻（孙氏手术）权重极高
须完整记录体外循环时间和深低温停循环时长。

高权重MCC/CC

术后低心排综合征（须IABP/ECMO辅助记录）
体外循环后急性肾功能衰竭需CRRT
术后严重心律失常需电复律或再次手术
合规警示：体外循环时间、停循环时间、搭桥数目须与麻醉单、手术护理记录三方一致。
神经内科DRG核心提示：脑卒中类型、部位和是否介入治疗是分组关键，溶栓/取栓操作权重远高于内科保守治疗。

神经内科主诊规则

脑梗死 → 区分部位（前循环/后循环）和病因（TOAST分型）
大动脉粥样硬化型/心源性/小血管病型编码不同；静脉溶栓（tPA）和血管内取栓为高权重操作，须有时间窗记录。
脑出血 → 区分硬膜下/硬膜外/蛛网膜下腔/脑实质出血
动脉瘤破裂SAH须有CTA/DSA证据；外科手术（开颅/介入栓塞）权重远高于保守。
癫痫持续状态 → 区分惊厥性/非惊厥性，脑电图监测记录必须完整
难治性癫痫持续状态须有ICU记录和多线治疗依据。
合规警示：溶栓治疗须有发病到用药时间记录（时间窗内），且与护理记录一致；取栓手术须有术前CTA/MRA证据。
神经外科DRG核心提示：开颅手术与内镜/介入操作分属不同ADRG，手术入路和肿瘤良恶性影响权重显著。

神经外科主诊及手术规则

颅内肿瘤 → 区分良性/恶性、部位、手术切除程度
全切/次全切/活检权重不同；立体定向活检单独编码；恶性肿瘤权重高于良性同部位。
脑出血/颅内血肿 → 区分高血压性/外伤性/动脉瘤性，手术/保守分组
动脉瘤夹闭或介入栓塞权重远高于保守；外伤性开颅清除血肿须区分硬膜下/外/脑内。
脊柱脊髓手术 → 神经外科主诊通常为脊髓/神经根压迫（G系列）
与骨科骨折分组不同，权重可更高。须有神经损害的影像和查体依据。
合规警示：颅内肿瘤切除须有术后病理报告支持良恶性诊断；手术切除程度须与影像学（MRI）结论一致。
呼吸科DRG核心提示：呼吸衰竭分型和通气方式是分组关键，有创通气属MCC最高权重项。

呼吸科主诊规则

肺炎 → 区分细菌性/病毒性/真菌性，有病原学结果时使用合并编码
重症肺炎（需呼吸支持）权重显著高于普通肺炎；须有病原学检查记录。
COPD急性加重 → 标注严重程度（GOLD分级）和触发因素
合并Ⅱ型呼吸衰竭属MCC；无创/有创通气须有血气分析和通气参数记录。
呼吸衰竭 → 须区分Ⅰ型（低氧）/Ⅱ型（高碳酸），有创通气为MCC
有创机械通气>96小时属极高MCC；间质性肺病急性加重须有HRCT依据。
肺栓塞 → 区分大面积/次大面积/低危，溶栓治疗为高权重操作
大面积肺栓塞须有心脏超声RV/LV比和休克记录；溶栓须有禁忌证评估记录。
合规警示：有创通气使用时长须与护理记录和麻醉/重症医学记录完全一致，不得虚报天数。
消化内科DRG核心提示：内镜操作类型是分组关键，治疗性内镜权重远高于诊断性；消化道出血须明确来源和严重程度。

消化内科主诊及操作规则

消化道出血 → 明确部位（上/下消化道）、出血量及止血方式
内镜止血（套扎/硬化/钛夹）比保守治疗权重高；需输血者须记录输血量及血红蛋白值。
ERCP → 区分诊断性/治疗性（取石/引流/括约肌切开）
治疗性ERCP权重约为诊断性2倍，取石数目和操作难度须在操作记录中体现。
肝硬化 → 必须标注合并症（腹水/食管胃底静脉曲张/肝性脑病/肝肾综合征）
MCC/CC依据ICD编码认定：肝性脑病（K72.x）属MCC；食管胃底静脉曲张破裂出血（I85.0）属MCC；腹水（R18）属CC。注意：Child-Pugh评分本身不是ICD编码项，须转化为具体合并症编码才能影响DRG分组。门静脉高压TIPS为高权重操作。
急性重症胰腺炎 → 须有CT严重度指数（CTSI评分）和器官衰竭记录
合并感染性坏死须有穿刺引流或外科干预记录，属极高权重组。
合规警示：ERCP操作并发症（急性胰腺炎）须有血淀粉酶结果和治疗记录，不得预防性填写。
普外科DRG核心提示：手术方式（开腹/腹腔镜/机器人）和恶性肿瘤根治程度直接影响分组；急诊手术与择期手术权重有差异。

普外科主诊及手术规则

胃肠肿瘤手术 → 区分根治切除/姑息切除，明确淋巴结清扫范围
D2根治术须有清扫数目记录；联合脏器切除（如胰体尾+脾）权重显著更高。
胰十二指肠切除术（Whipple）→ 属极高权重组
须明确是经典Whipple还是保留幽门术式；合并血管重建权重更高。
急腹症 → 穿孔/绞窄/肠梗阻须如实记录，影响MDC分配
阑尾炎穿孔腹膜炎权重远高于单纯阑尾炎；腹腔镜手术中转开腹须按实际完成术式编码。
合规警示：腹腔镜手术中转开腹须在手术记录中注明，编码应按实际完成的手术方式填写。
胸外科DRG核心提示：肺癌手术切除范围（楔形/肺叶/全肺）和手术方式（VATS/开胸/机器人）影响权重。

胸外科主诊及手术规则

肺癌手术 → 区分楔形切除/肺段切除/肺叶切除/全肺切除，明确淋巴结清扫
肺叶切除+纵隔淋巴结清扫（根治术）权重最高；VATS/机器人辅助须单独编码。
食管癌手术 → 区分颈/胸/腹三切口，管状胃成形，重建方式
McKeown术式（三切口）权重最高；术后吻合口瘘须有消化道造影或内镜记录，属MCC。
纵隔肿瘤/胸腺瘤 → 须有病理分型（WHO分型）和Masaoka分期
合并重症肌无力须有AchR抗体和肌电图记录；胸腺癌与胸腺瘤编码不同，权重差异大。
合规警示：肺切除范围须与术后胸部CT和病理报告一致；淋巴结清扫数目须与病理报告相符。
血管外科DRG核心提示：腔内手术与开放手术分属不同ADRG，主动脉手术须区分解剖位置和腔内/开放方式。

血管外科主诊及手术规则

主动脉瘤 → 区分EVAR（腔内修复）/开放手术，胸/腹主动脉瘤须区分
破裂主动脉瘤急诊手术为最高权重组；须有人工血管记录。
下肢动脉闭塞 → 区分腔内治疗（PTA/支架）/搭桥手术/截肢
股腘动脉旁路移植权重高于PTA；膝下截肢须记录术前坏死/感染范围，有无血运重建。
深静脉血栓/肺栓塞 → 区分导管溶栓/机械取栓/下腔静脉滤器/系统溶栓
导管接触性溶栓（CDT）权重高于系统溶栓；大面积肺栓塞须有心脏超声RV/LV比和休克记录。
合规警示：腔内器材（支架型血管/支架）规格须与耗材使用台账一致；截肢须有充分的不可逆缺血依据记录。
泌尿外科DRG核心提示：肿瘤根治手术与腔内手术分属不同高权重组，结石手术须区分微创方式。

泌尿外科主诊及手术规则

前列腺癌 → 区分根治性前列腺切除（开放/腹腔镜/机器人辅助）与内分泌治疗入院
机器人辅助根治性切除术须单独编码，权重高于腹腔镜；内分泌治疗入院以Z51系列为主诊。
肾癌 → 区分根治性肾切除/保留肾单位手术（肾部分切除）
肾部分切除保肾手术权重与根治相近，须注明肿瘤大小和切缘情况。
尿路结石 → 区分ESWL/输尿管镜碎石取石/PCNL（经皮肾镜）
PCNL权重最高，须有结石大小、位置和手术时间记录。
膀胱癌 → 区分TURBT（经尿道切除）与根治性膀胱切除+尿流改道
根治性膀胱切除+原位新膀胱术为极高权重，须记录淋巴结清扫情况。
合规警示：腔内手术须与术前影像（CTU/IVU/KUB）结果一致；结石大小、位置须与手术记录相符。
妇科DRG核心提示：恶性肿瘤根治手术与良性疾病手术权重差异大；腹腔镜/开腹/宫腔镜/机器人须分别编码。

妇科主诊及手术规则

宫颈癌/子宫内膜癌/卵巢癌 → 区分手术分期、根治程度、有无淋巴结清扫
广泛性子宫切除+盆腔淋巴结清扫（根治术）权重最高；同期肠切除/膀胱切除需记录联合手术。
子宫肌瘤/腺肌症 → 区分肌瘤切除/子宫切除（全切/次全切）、有无附件切除
腹腔镜子宫次全切除与全切除编码不同；子宫重量须在手术记录中注明。
宫腔镜手术 → 区分诊断性/治疗性
宫腔镜下宫腔粘连重度分离权重高，须有宫腔镜评分（AFS分级）记录。
合规警示：妇科恶性肿瘤手术须有术前肿瘤标志物、术中冷冻切片及术后石蜡病理支持分期诊断。
产科DRG核心提示：分娩方式（阴道分娩/剖宫产）是分组主轴，合并症和新生儿情况影响权重。

产科主诊及分娩规则

分娩方式须精确记录：自然分娩/助产（产钳/胎吸）/择期剖宫产/急诊剖宫产
急诊剖宫产权重高于择期；合并前置胎盘、胎盘植入剖宫产为高权重，须有超声依据。
妊娠合并症 → 子痫前期/子痫须标注重度，妊娠期糖尿病须区分GDM/糖尿病合并妊娠
重度子痫前期（收缩压≥160mmHg或有器官损伤）属MCC；子痫发作须有发作记录。
产后并发症 → 产后出血须记录出血量（≥500ml），处理措施须完整
需子宫切除止血的产后出血为极高权重；DIC须有凝血功能检查结果支持。
合规警示：剖宫产指征须在病历中有明确记录（与手术同意书一致），无指征剖宫产属不合规手术。
儿科DRG核心提示：年龄（新生儿/婴幼儿/儿童）是分组重要因素，新生儿疾病独立进入P-MDC系列，权重普遍偏高。

儿科主诊及年龄分组规则

新生儿疾病 → 独立进入P系列，须明确胎龄（早产/足月）
极低体重儿（<1500g）和超低体重儿（<1000g）权重极高；呼吸窘迫综合征须有肺表面活性物质治疗记录。
小儿肺炎 → 区分病原体（细菌/病毒/支原体）和严重程度
重症肺炎（需呼吸支持）属MCC；合并心衰、脓胸须有相应检查治疗记录。
先天性心脏病手术 → 极高权重，须区分手术类型（根治/姑息/分期）
体外循环下根治手术权重最高；Fontan/Norwood手术须完整记录血流动力学数据。
合规警示：儿科患者年龄（出生日期）须与病历首页准确一致，新生儿须精确到天数，影响分组。
内分泌科DRG核心提示：糖尿病合并症是分组核心，须精确标注急性并发症类型和严重程度。

内分泌科主诊规则

糖尿病急性并发症 → DKA/HHS须区分，明确有无昏迷
糖尿病酮症酸中毒（DKA）伴昏迷权重远高于不伴昏迷；HHS须有血糖、血酮、渗透压记录。
糖尿病慢性并发症 → 须精确标注类型和分期
糖尿病肾病IV期以上属高权重CC；糖尿病足须标注Wagner分级，III级以上为MCC，须有血管评估和清创记录。
甲状腺危象 → 须有Burch-Wartofsky评分≥45分和ICU记录
甲状腺危象属MCC，须有ICU监护或多脏器支持记录方可填报。
垂体腺瘤/肾上腺肿瘤 → 手术（经鼻蝶/腹腔镜肾上腺切除）权重远高于保守
库欣综合征合并高血压/糖尿病/骨质疏松属高权重CC组合；须有激素分泌过多的实验室依据。
合规警示：糖尿病分型（1型/2型）须有依据；并发症分期须与检验结果（HbA1c、尿蛋白定量、眼底检查）一致。
肾内科DRG核心提示：透析操作类型和CKD分期是分组关键；肾穿刺活检和急性肾损伤严重程度须精确记录。

肾内科主诊及操作规则

AKI → 须标注KDIGO分期（1/2/3期）和病因
AKI 3期（无尿 + Cr >3倍基线）属MCC；需CRRT者权重显著提升，须记录CRRT时长。
CKD → 须标注分期（G1~G5）和病因（原发性/继发性）
CKD 5期（尿毒症）需透析者进入高权重组；腹膜透析与血液透析分属不同ADRG。
肾穿刺活检 → 须有病理报告，依据病理结果完善诊断编码
肾穿刺操作本身为诊断性操作（低权重），但病理确诊后的疾病编码（IgA/MN/FSGS等）影响分组。
肾病综合征 → 必须标注有无MCC/CC（血栓/感染/AKI）
肾病综合征合并肾静脉血栓（CT/MRI证实）属MCC；合并重症感染属高MCC。
合规警示：CRRT与间歇性血液透析须分别编码，使用时长须与护理记录一致；腹膜透析须有管路记录。
血液科DRG核心提示：化疗方案、造血干细胞移植（HSCT）类型是最高权重分组因素；血液病诊断须有骨髓/病理依据。

血液科主诊及操作规则

造血干细胞移植 → 区分异基因/自体移植，清髓/非清髓预处理
异基因HSCT权重极高（RW >10）；移植后GVHD须有分级记录（I~IV级），重度GVHD属MCC。
急性白血病 → 区分ALL/AML，明确诱导化疗方案
初次诱导化疗权重高于巩固维持；APL须记录亚砷酸/ATRA使用，分化综合征须有治疗记录。
淋巴瘤 → 区分霍奇金/非霍奇金，侵袭性/惰性，化疗方案
含利妥昔单抗方案须有CD20表达检测记录；CAR-T治疗须有CRS分级监测记录。
粒缺热 → 须明确感染来源，记录G-CSF使用时长
粒缺热合并感染性休克属MCC；血培养阳性须有药敏结果和抗生素调整记录。
合规警示：血液病诊断须有骨髓穿刺/活检、免疫分型、染色体、基因检测等结果支持，不可仅凭临床判断填报。
风湿免疫科DRG核心提示：自身免疫病须明确脏器受累，生物制剂/靶向药物使用和脏器损伤程度影响权重。

风湿免疫科主诊规则

SLE → 须明确活动度（SLEDAI评分）和脏器受累
狼疮性肾炎（须有肾活检分型）、神经精神狼疮、狼疮肺炎属MCC，须有脏器受累的实验室和影像依据。
RA → 区分活动度（DAS28评分），明确生物制剂/JAK抑制剂使用适应证
合并间质性肺炎（RA-ILD）属高权重CC；生物制剂使用须有关节评分和影像学进展依据。
ANCA相关性血管炎 → 须明确受累血管/器官
肾受累（ANCA相关性肾炎）合并AKI须有肾活检结果；重度血管炎须有血管影像学依据。
合规警示：生物制剂治疗须有适应证记录（活动度评分、传统药物失败）；免疫抑制剂剂量须与护理记录和费用清单一致。
感染科DRG核心提示：病原体明确与否影响分组，脓毒血症须依据Sepsis-3标准，抗病毒治疗须有病毒学依据。

感染科主诊规则

病毒性肝炎 → 须明确分型和临床分型（急性/慢性/肝衰竭）
慢性乙肝肝衰竭须有Child-Pugh/MELD评分，合并肝性脑病属MCC。
脓毒血症 → 须符合Sepsis-3标准（感染 + qSOFA/SOFA评分）
脓毒性休克须有血压记录（需升压药维持）+ 乳酸升高（>2mmol/L），属最高MCC。
结核病 → 区分肺结核/肺外结核/耐药结核，须有病原学或病理依据
耐多药结核（MDR-TB）权重高于敏感结核；中枢神经系统结核属MCC。
合规警示：感染性疾病诊断须有病原学证据（培养、PCR、抗原/抗体），无依据的病原体诊断属高套风险。
重症医学科DRG核心提示：生命支持技术（有创通气/ECMO/CRRT/IABP）是最高权重分组因素，须有使用时长和适应证记录。

ICU主诊及操作规则

有创机械通气 → 须记录插管时间、通气天数、撤机时间
有创通气 >96小时属极高MCC，须与麻醉单、呼吸机记录一致；ARDS须有P/F比值记录（Berlin标准）。
ECMO → 区分V-A（心源性休克/心脏骤停后）与V-V（呼吸衰竭）
ECMO使用时长须与灌注师记录、医嘱、护理记录三方一致；撤机时间须明确记录。
MODS → 须明确受累器官（≥2个），记录SOFA评分
SOFA评分≥15属极高权重；须有每个受累器官的评分依据。
合规警示：ICU是医保稽查重点，生命支持设备使用时长须与护理记录和医嘱记录完全吻合，不得虚报使用天数。
眼科DRG核心提示：手术类型（白内障/青光眼/视网膜/眼眶）是分组主轴，双眼单次住院须明确编码规则。

眼科主诊及手术规则

白内障手术 → 区分超声乳化/囊外摘除，明确人工晶体类型
双眼同期手术须标注双侧；飞秒激光辅助须单独编码；合并青光眼联合手术权重更高。
视网膜脱离手术 → 区分巩膜扣带/玻璃体切除/气体/硅油填充
玻璃体切除+硅油填充权重最高；PVR C级以上属MCC，须有术前广角眼底照相记录。
青光眼 → 区分小梁切除/房水引流物植入/激光治疗
引流阀植入（Ahmed/Baerveldt）权重高于小梁切除；急性闭角型青光眼发作须有眼压记录。
合规警示：眼科手术耗材（人工晶体型号、引流阀品规）须与费用清单和手术记录一致；双眼同期手术须符合单次住院编码规范。
耳鼻喉科DRG核心提示：头颈肿瘤根治手术与功能性手术权重差异大，须精确区分手术范围和修复重建方式。

耳鼻喉科主诊及手术规则

喉癌/下咽癌手术 → 区分部分切除/全喉切除/根治性颈清扫
全喉切除+颈清扫权重最高；修复重建（游离皮瓣/肌皮瓣）须单独编码，联合手术权重更高。
鼻腔鼻窦手术 → 区分FESS范围（单窦/多窦/全组）
全组鼻窦开放（Draf III）权重高于单窦手术；合并颅底侵犯者须有影像学支持。
人工耳蜗植入 → 须有听力评估（ABR/ASSR）和CT/MRI提示耳蜗形态正常
人工耳蜗属极高值耗材，权重极高；须有听力学委员会评估记录。
合规警示：头颈肿瘤手术的颈清扫范围须与术中记录和病理报告的淋巴结数目一致。
精神科DRG核心提示：精神科独立进入U-MDC系列，住院目的（急性期/慢性维持）和合并躯体疾病是分组关键。

精神科主诊规则

精神分裂症 → 须标注发作类型（首次/复发/难治）、严重程度（PANSS评分）
首次发作住院权重高于慢性维持期；难治性须有两种以上抗精神病药物足量足疗程失败记录。
双相情感障碍 → 区分躁狂/抑郁/混合发作，明确严重程度
伴精神病性症状的躁狂发作权重高于不伴；须有YMRS/HDRS量表评分记录。
MECT（改良电休克治疗）→ 须有治疗次数、麻醉记录和适应证
须有适应证（难治性抑郁/危及生命的精神症状）；不得以MECT次数虚高权重。
精神科合并躯体疾病 → 合并症须有积极治疗证据，否则不得列为CC
须有专科会诊记录和治疗措施。
合规警示：精神科诊断须有精神科专科评估和诊断量表依据；不得将日常监测性住院填报为急性发作期。
康复科DRG核心提示：康复病种须有明确原发病，康复治疗方案和功能评定是分组依据，FIM评分直接影响分组。

康复科主诊及评定规则

脑卒中后康复 → 须有入院FIM评分，主诊为后遗症（I69系列）
FIM评分≤40（重度功能障碍）权重高于轻度；须记录入院/出院FIM变化值作为康复疗效证明。
骨折后康复 → 主诊须为骨折愈合阶段或功能障碍，而非原始骨折编码
关节置换术后康复须标注置换类型和手术日期；须有关节活动度测量记录（ROM）。
脊髓损伤康复 → 须有ASIA分级（A/B/C/D/E），明确损伤平面
完全性脊髓损伤（ASIA A级）权重高于不完全性；须有神经功能评定和膀胱管理记录。
康复治疗项目 → PT/OT/ST须有每日治疗记录，次数与费用清单一致
不得虚填康复治疗次数；FIM评分须由专业康复治疗师完成并签字。
合规警示：康复科是医保稽查重点，康复治疗项目须有每日治疗记录；FIM评分须由专业康复治疗师完成并签字。
CHS-DRG 2.0 核心分组三步法
1 MDC 大类（主诊断决定）
26个MDC大类，由主要诊断ICD-10编码唯一决定。主诊选错 = MDC错 = 整个分组全部错误。
2 ADRG 核心组（主诊 + 手术）
主诊断 + 主要手术/操作共同决定进入哪个核心DRG组（ADRG）。2.0版共409个ADRG，较1.0版增加33组。
3 DRG 细分组（CC/MCC细分）
其他诊断中的严重合并症（MCC）→ 高权重组；普通合并症（CC）→ 中权重组；无CC/MCC → 低权重组。
支付标准计算公式
支付标准 = DRG权重 × 费率
权重：该病组例均费用 ÷ 全部病例例均费用（反映资源消耗相对程度）
费率：各地不同。北京2025年费率20,425元；四川省采用点数法付费，基准点数和系数由各市（州）确定
示例：若病组权重 = 2.5，则支付标准 ≈ 2.5 × 20,425 = 51,063 元
主要诊断"三最原则"
危害最大 首选
本次住院对患者健康危害程度最重的疾病
消耗最多
消耗药品、耗材、检查等医疗资源最多的疾病
住院最长
导致住院时间延长最多的疾病诊断
特例单议申报条件（2025版规程）
① 住院费用超过病组支付标准 3倍及以上
② 使用创新药品、创新医疗技术或高值耗材
③ 多学科联合诊疗（MDT）或复杂手术转科病例
④ 现行分组方案未能涵盖的罕见病例
申报比例：原则上不超过DRG出院总病例的 5%
⚠️ 合规底线：诊断排序须真实反映临床诊疗过程。通过虚增诊断、人为调整主诊高套权重属违规，医保稽核将追回款项并处罚。
手术操作对权重的影响（参考值）
内科保守治疗（无手术） RW 0.5 ~ 1.5
无手术操作编码，进入内科DRG组，权重最低。MCC/CC可提升至1.5左右。
一般手术（阑尾切除/腹腔镜胆囊等） RW 1.5 ~ 2.5
常见择期手术，进入外科DRG组，权重约为内科组2倍。合并MCC可升至3.0。
中等手术（关节置换/PCI单支/肿瘤根治等） RW 2.5 ~ 4.5
手术难度中等，技术要求高，是大多数科室的主要高权重分组区间。
复杂手术（多节段脊柱融合/多支血管PCI等） RW 4.5 ~ 8.0
技术难度大、耗材多、手术时间长，须有完整操作记录和内固定/耗材使用记录。
极复杂手术（TAVR/ECMO辅助/多脏器联合切除等） RW 8.0 ~ 15.0+
顶级术式，需多学科团队、高值耗材、完整评估记录，是医院DRG最高权重来源。
