骨科DRG核心提示：主要诊断决定进入I系列（肌肉骨骼系统）还是B系列（神经系统），影响权重差异可达数倍。

主诊选择关键规则

多部位损伤 → 须区分情况，不可一律"神经优先"
脊柱骨折合并脊髓损伤（S14/S24/S34）→ 脊髓损伤做主诊，B系列权重更高；四肢骨折合并外周神经损伤 → 通常仍选骨折（I系列），外周神经损伤为CC。判断原则：比较两种主诊路径对应的DRG权重，选权重更高者。
脊柱手术 → 区分颈/胸/腰段及前路/后路
颈椎前路/后路手术编码不同，影响ADRG分配，须精确编码到手术入路。
关节置换 → 明确单/双侧、初次/翻修
翻修（revision）权重显著高于初次置换；双侧同次住院须标注双侧。
合并症加权：骨科高权重MCC/CC
DVT（I82.x）、肺栓塞（I26.x）、伤口感染、脂肪栓塞（T79.1）、ARDS须有治疗记录方可填报。

常见骨科高权重病组示例

脊柱融合术（多节段）
伴MCC权重 > 5.0，需记录融合节段数及内固定情况。
髋关节翻修
翻修手术权重约为初次置换1.5~2倍，须有明确翻修指征记录。
复杂骨盆骨折
合并血管损伤或腹腔脏器损伤时，主诊可考虑血管损伤（T系列），须多科会诊记录。
合规警示：骨科是医保稽核重点科室。不得仅凭骨折加上神经损伤编码而无神经科相关检查或治疗记录，否则属高套违规。
肿瘤科DRG核心提示：住院目的不同，主诊断选择规则完全不同——手术、放疗、化疗、支持治疗各有专属原则。

肿瘤主诊断选择四大场景

场景1 手术切除为主要目的
主诊选恶性肿瘤本身（如胃恶性肿瘤 C16），并记录具体手术方式（根治/姑息切除）。
场景2 单纯化疗/靶向治疗
主诊选"恶性肿瘤化学治疗"（Z51.1），原发肿瘤作为其他诊断。
场景3 单纯放疗
主诊选"恶性肿瘤放射治疗"（Z51.0），与化疗同次住院时以消耗资源多者为主诊。
场景4 并发症/急症处理
因并发症（如肠梗阻、上消化道出血）入院，主诊选并发症；原发肿瘤为其他诊断。可显著提升权重。

肿瘤科高权重MCC/CC

恶性胸腔积液、恶性腹腔积液（须有穿刺引流记录）
化疗后Ⅳ度骨髓抑制（须有血常规结果 + 治疗记录）
肿瘤溶解综合征、弥漫性血管内凝血（DIC）
脓毒血症/感染性休克（须有培养结果或影像支持）
合规警示：不得在所有化疗患者中统一加注"恶心呕吐"诊断。须有相应治疗记录方可作为CC填报。
心内科DRG核心提示：介入手术类型（PCI、起搏器植入、射频消融等）直接决定ADRG分组，比内科诊断权重高2~5倍。

心内科主诊选择规则

急性心肌梗死 → 明确部位与类型
STEMI前壁I21.0、下壁I21.1；NSTEMI统一I21.4，不按部位细分。有PCI时主诊仍选AMI，手术操作填PCI。
心衰 → 明确NYHA分级 + 类型
收缩性（I50.0）/舒张性（I50.1）编码不同；NYHA Ⅲ-Ⅳ级属MCC。
介入手术类型须精确
单支/多支血管PCI、DES/BMS支架、CTO病变须细分编码；多支血管病变权重更高。

心内科高权重MCC/CC

心源性休克（须有血压/血流动力学记录 + 升压药使用）
完全性房室传导阻滞（须有心电图记录）
急性肾衰竭（Cr升高 + 少尿，须记录透析或利尿治疗）
恶性心律失常：室颤/室速需电复律（须有除颤记录）
合规警示：心衰NYHA分级须与病程记录一致；PCI支架数量须与手术记录及耗材台账一致。
心外科DRG核心提示：手术类型和是否体外循环是分组关键，搭桥血管数目、瓣膜置换/成形方式须精确记录。

心外科主诊及手术规则

CABG → 明确体外循环/非体外循环、桥血管数目
体外循环CABG权重高于非体外，多支搭桥权重更高；搭桥数目须与手术记录一致。
瓣膜手术 → 区分置换/成形、单瓣/多瓣、生物瓣/机械瓣
双瓣置换权重远高于单瓣，合并CABG的联合手术属最高权重组，须有超声心动图支持。
主动脉手术 → 区分根部/升主动脉/弓部，全弓替换+支架象鼻（孙氏手术）权重极高
须完整记录体外循环时间和深低温停循环时长。

高权重MCC/CC

术后低心排综合征（须IABP/ECMO辅助记录）
体外循环后急性肾功能衰竭需CRRT
术后严重心律失常需电复律或再次手术
合规警示：体外循环时间、停循环时间、搭桥数目须与麻醉单、手术护理记录三方一致。
神经内科DRG核心提示：脑卒中类型、部位和是否介入治疗是分组关键，溶栓/取栓操作权重远高于内科保守治疗。

神经内科主诊规则

脑梗死 → 区分部位（前循环/后循环）和病因（TOAST分型）
静脉溶栓（tPA）和血管内取栓为高权重操作，须有时间窗记录。
脑出血 → 须精确区分四类编码，不得混用
蛛网膜下腔出血I60.x / 脑实质出血I61.x / 硬膜下血肿I62.0 / 硬膜外血肿I62.1——四类编码不同，入组不同，动脉瘤夹闭或栓塞权重极高。
癫痫持续状态 → 区分惊厥性/非惊厥性，脑电图监测记录必须完整
难治性癫痫持续状态须有ICU记录和多线治疗依据。
合规警示：溶栓治疗须有发病到用药时间记录（时间窗内），且与护理记录一致；取栓手术须有术前CTA/MRA证据。
神经外科DRG核心提示：开颅手术与内镜/介入操作分属不同ADRG，手术入路和肿瘤良恶性影响权重显著。

神经外科主诊及手术规则

颅内肿瘤 → 区分良性/恶性、部位、手术切除程度
全切/次全切/活检权重不同；立体定向活检单独编码；恶性肿瘤权重高于良性同部位。
脑出血/颅内血肿 → 区分高血压性/外伤性/动脉瘤性
动脉瘤夹闭或介入栓塞权重远高于保守；外伤性开颅清除血肿须区分硬膜下/外/脑内。
脊柱脊髓手术 → 神经外科主诊通常为脊髓/神经根压迫（G系列）
与骨科骨折分组不同，权重可更高。须有神经损害的影像和查体依据。
合规警示：颅内肿瘤切除须有术后病理报告支持良恶性诊断；手术切除程度须与影像学（MRI）结论一致。
呼吸科DRG核心提示：有创通气属MCC最高权重项。【禁区】呼吸衰竭（J96.x）禁止作主诊——进入MDCA先期分组（有创通气≥96h触发）与主诊选择是两件独立的事，进入MDCA后主诊仍须选原发病。

呼吸科主诊规则

肺炎 → 区分细菌性/病毒性/真菌性，有病原学结果时使用合并编码
重症肺炎须有呼吸支持记录；须有病原学检查记录。
COPD急性加重 → 标注严重程度（GOLD分级）和触发因素
合并Ⅱ型呼吸衰竭属MCC；无创/有创通气须有血气分析记录。
肺栓塞 → 区分大面积/次大面积/低危，溶栓治疗为高权重操作
大面积肺栓塞须有心脏超声RV/LV比和休克记录。
合规警示：有创通气使用时长须与护理记录和重症医学记录完全一致，不得虚报天数。
消化内科DRG核心提示：治疗性内镜权重远高于诊断性；肝硬化合并症须转化为具体ICD编码，Child-Pugh评分本身不影响DRG分组。

消化内科主诊及操作规则

消化道出血 → 明确部位、出血量及止血方式
内镜止血比保守治疗权重高；需输血者须记录输血量及血红蛋白值。
ERCP → 区分诊断性/治疗性（取石/引流/括约肌切开）
治疗性ERCP权重约为诊断性2倍。
肝硬化 → 合并症须转化为ICD编码方可影响分组
肝性脑病（K72.x）属MCC；食管胃底静脉曲张破裂出血（I85.0）属MCC；腹水（R18）属CC。Child-Pugh评分须转化为具体合并症编码。
急性重症胰腺炎 → 须有CTSI评分和器官衰竭记录
合并感染性坏死须有穿刺引流或外科干预记录。
合规警示：ERCP操作并发症（急性胰腺炎）须有血淀粉酶结果和治疗记录，不得预防性填写。
普外科DRG核心提示：手术方式和恶性肿瘤根治程度直接影响分组；急诊手术与择期手术权重有差异。

普外科主诊及手术规则

胃肠肿瘤手术 → 区分根治切除/姑息切除，明确淋巴结清扫范围
D2根治术须有清扫数目记录；联合脏器切除权重显著更高。
胰十二指肠切除术（Whipple）→ 属极高权重组
须明确是经典Whipple还是保留幽门术式；合并血管重建权重更高。
急腹症 → 穿孔/绞窄须如实记录
穿孔腹膜炎权重远高于单纯炎症；腹腔镜手术中转开腹须按实际完成术式编码。
合规警示：腹腔镜手术中转开腹须在手术记录中注明，编码应按实际完成的手术方式填写。
胸外科DRG核心提示：肺癌切除范围（楔形/肺叶/全肺）和手术方式（VATS/开胸/机器人）影响权重。

胸外科主诊及手术规则

肺癌手术 → 区分切除范围，明确淋巴结清扫
肺叶切除+纵隔淋巴结清扫（根治术）权重最高；VATS/机器人辅助须单独编码。
食管癌手术 → 区分颈/胸/腹三切口
McKeown术式权重最高；术后吻合口瘘须有消化道造影或内镜记录，属MCC。
纵隔肿瘤/胸腺瘤 → 须有WHO病理分型和Masaoka分期
合并重症肌无力须有AchR抗体记录；胸腺癌与胸腺瘤编码不同，权重差异大。
合规警示：肺切除范围须与术后胸部CT和病理报告一致；淋巴结清扫数目须与病理报告相符。
血管外科DRG核心提示：腔内手术与开放手术分属不同ADRG，主动脉手术须区分解剖位置和腔内/开放方式。

血管外科主诊及手术规则

主动脉瘤 → 区分EVAR/开放手术，胸/腹主动脉须区分
破裂主动脉瘤急诊手术为最高权重组；须有人工血管记录。
下肢动脉闭塞 → 区分腔内治疗/搭桥/截肢
截肢须记录截肢平面；PTA/支架置入须与耗材台账一致。
深静脉血栓 → 区分导管溶栓/机械取栓/滤器/系统溶栓
CDT（导管接触溶栓）权重更高；须有术前超声证据。
合规警示：腔内器材规格须与耗材台账一致；血管造影须区分诊断性/治疗性编码。
泌尿外科DRG核心提示：手术方式（开放/腹腔镜/机器人/内镜）影响ADRG分组；肿瘤分期须有病理支持。

泌尿外科主诊及手术规则

前列腺癌根治术 → 区分开放/腹腔镜/机器人辅助
机器人辅助须单独编码；须有PSA和穿刺活检病理支持分期。
肾癌手术 → 区分根治性肾切除/肾部分切除，注明肿瘤大小
保肾手术（肾部分切除）与根治性切除编码不同。
尿路结石手术 → 区分ESWL/输尿管镜/PCNL
PCNL权重最高；须有术前影像学石头大小、位置记录。
根治性膀胱切除+尿流改道 → 极高权重
须记录淋巴结清扫范围；尿流改道方式须与手术记录一致。
合规警示：机器人辅助手术须有专项记录，不得以腹腔镜手术编码替代。
妇科DRG核心提示：恶性肿瘤手术须明确分期、淋巴结清扫范围；子宫切除须区分全切/次全切及手术方式。

妇科主诊及手术规则

宫颈癌/内膜癌/卵巢癌根治 → 须明确分期和淋巴结清扫
恶性肿瘤须有术前肿瘤标志物、术中冷冻切片和术后病理支持。
子宫切除 → 区分全切/次全切、腹腔镜/开腹/机器人
有无附件切除须一并记录；机器人辅助须单独编码。
宫腔镜 → 区分诊断性/治疗性
治疗性宫腔镜权重更高；须有具体操作记录（切除/电切/消融等）。
合规警示：妇科恶性肿瘤须有病理报告支持编码，不得凭影像学描述单独编恶性编码。
产科DRG核心提示：分娩方式和孕周是分组关键。【重要】母婴必须分开建档：新生儿诊断（黄疸/脐炎等）绝对不可混入产妇病案，否则产妇入组错误。

产科主诊规则

分娩方式须精确 → 自然分娩/助产/择期剖宫产/急诊剖宫产
急诊剖宫产权重高于择期；编码不同，权重差异显著。
重度子痫前期（O14.1）→ MCC
须满足：收缩压≥160mmHg 或靶器官损伤（蛋白尿+器官损害）。
产后出血量阈值
阴道分娩≥500ml / 剖宫产≥1000ml 才可编O72.x；子宫切除止血（O71.3）为极高权重，须有手术记录。
妊娠期糖尿病须区分
GDM（O24.4）vs 糖尿病合并妊娠（O24.0~O24.3），编码不同。
胎盘植入/前置胎盘须有影像依据
胎盘植入（O43.2）须有超声/MRI；早产须记录孕周，<28周极早产权重最高。
合规警示：新生儿必须独立建立病案（MDCP分组），不可将新生儿诊断混入产妇病案。产后出血量须与麻醉单和护理记录一致。
儿科DRG核心提示：新生儿（<29天）独立进入P系列；胎龄和出生体重是分组关键参数。

儿科主诊规则

新生儿疾病 → 进入P系列，胎龄和体重决定分组
极低体重儿（<1500g）和超低体重儿（<1000g）权重极高；年龄须精确到天数。
先天性心脏病手术 → 区分根治/姑息/分期
根治术权重最高；须有超声心动图诊断支持。
重症肺炎 → 区分细菌性/病毒性，有无呼吸支持
有创通气属MCC，权重显著提升；须有病原学证据。
合规警示：儿科年龄须与病历首页精确一致，新生儿精确到天数；体重须有出生时记录支持。
内分泌科DRG核心提示：糖尿病编码须精确到并发症类型和严重程度；CKD分期须标注，直接影响DRG分组。

内分泌科主诊规则

DKA → 区分有无昏迷
E11.0（有昏迷）vs E11.1（无昏迷）权重显著不同；须有血糖/血酮/血气pH记录。
糖尿病肾病 → 双重编码 + CKD分期
E11.2↑N08.3*，须同时标注CKD分期（N18.x）；CKD 4~5期属高权重CC。
糖尿病足 → 须标注Wagner分级
Wagner III级以上伴坏疽（E11.5+L97）属MCC；须有清创/截肢手术记录。
甲状腺危象（E05.5）→ 须有Burch-Wartofsky评分≥45分和ICU记录
垂体/肾上腺手术权重远高于保守治疗，须区分微创/开放术式。
合规警示：Mogensen分期须转化为CKD分期（N18.x）才能影响DRG分组；糖尿病足Wagner分级须有查体记录支持。
肾内科DRG核心提示：AKI须标注KDIGO分期；CRRT使用时长须与护理记录完全吻合。

肾内科主诊规则

AKI → 须标注KDIGO分期（1/2/3期），3期属MCC
须有Cr基线值对比；须记录少尿时间。
CRRT → 使用时长须与护理记录一致，权重显著高于间歇透析
须记录CRRT模式（CVVH/CVVHDF等）和上机时间。
CKD → 须标注G1~G5分期，CKD 5期接受透析为高权重
肾穿刺活检后须依据病理结果完善诊断编码。
合规警示：CRRT时长须与护理单三方吻合（医嘱、护理记录、灌注记录）；AKI须有肌酐基线值，否则分期无据。
血液科DRG核心提示：造血干细胞移植为最高权重组；CAR-T治疗CRS分级是关键MCC；化疗权重主要由MCC/CC决定而非方案类型。

血液科主诊规则

造血干细胞移植 → 区分异基因（28.0x）/自体（41.0x）、清髓/非清髓
异基因移植权重极高；须有预处理方案记录。移植后GVHD须有分级（III~IV级为MCC），须有靶器官受累病理或临床依据。
急性白血病化疗 → 权重由MCC/CC决定，非方案类型
CHS-DRG 2.0不区分诱导/巩固/挽救方案；不得以"初次诱导"名义虚增标注。高权重MCC：粒缺伴感染性休克（A41.x+D70）、IV度骨髓抑制（D70）、DIC（D65）、侵袭性真菌感染。
CAR-T治疗 → CRS分级须记录（1~4级），CRS≥3级为高MCC
须有IL-6升高和器官损伤证据。注意：CHS-DRG 2.0对CAR-T分组尚在完善阶段，部分地区按特例单议处理，建议与医保管理部门确认。
合规警示：粒缺热须有血培养结果；感染性休克须有升压药记录和乳酸值；不得无依据标注IV度骨髓抑制。
风湿免疫科DRG核心提示：SLE须标注SLEDAI评分和受累脏器；合并间质性肺病须有HRCT依据属高权重CC。

风湿免疫科主诊规则

SLE → 须标注SLEDAI评分，狼疮性肾炎须肾活检病理分型
大剂量激素冲击须有适应证记录；浆细胞置换/静注丙球须有费用支持。
RA → 须有DAS28评分，生物制剂/JAK抑制剂须有传统药物失败记录
合并间质性肺病（HRCT依据）属高权重CC。
ANCA相关性血管炎 → 须有肾活检和ANCA结果
合并AKI属MCC，须有Cr变化记录。
合规警示：风湿科合并躯体疾病须有会诊记录和治疗措施，否则不得列为CC；生物制剂使用须有适应证评估记录。
感染科DRG核心提示：病原学证据是分组关键；脓毒血症须符合Sepsis-3标准，须有血压和乳酸值记录。

感染科主诊规则

病毒性肝炎 → 须明确分型，肝衰竭须有Child-Pugh/MELD评分
慢加急性肝衰竭（ACLF）权重最高；须有凝血和胆红素动态记录。
脓毒血症 → 须符合Sepsis-3标准
须有血压记录（需升压药维持）+ 乳酸>2mmol/L + 血培养结果。
结核病 → 区分肺结核/肺外/耐药
须有病原学或病理依据；耐药结核须有药敏结果。
合规警示：感染性疾病诊断须有病原学证据（培养/PCR/抗原/抗体），仅凭临床表现编写病原体编码属违规。
🚨 双禁区：呼吸衰竭（J96.x）和ARDS（有明确原发病时）均禁止作主诊！进入MDCA后主诊仍须选原发病。

引流阀植入权重更高；须有眼压记录支持手术适应证。

范围越广权重越高；须有术前CT支持。

须与耗材台账一致；单侧/双侧权重不同。

改良电休克权重高于药物治疗；须有知情同意记录。
精神科合并躯体疾病 → 须有会诊记录和治疗措施

康复科主诊规则

主诊须为后遗症/功能障碍（非原发病急性期编码）
如脑梗死恢复期用I69.3（脑梗死后遗症），而非I63.x（急性期）。

FIM分值须与评估量表记录一致。
脊髓损伤须有ASIA分级（A~E级）和损伤平面记录
A级（完全损伤）权重最高；须有神经科会诊记录。
康复治疗项目（PT/OT/ST）须有每日治疗记录
次数与费用清单须一致；吸入性肺炎须有体温和肺部影像记录。

DRG细分组
分组逻辑流程
第一步：先期分组（MDCA）优先
有创呼吸机≥96h / ECMO / 器官移植 → 无论主诊是什么都优先进入MDCA。但进入MDCA不代表可以忽略主诊选择——主诊仍须选原发病，影响ADRG细分。
第二步：特殊人群分组
新生儿（MDCP，年龄<29天）/ HIV（MDCY）/ 多发严重创伤（MDCZ）→ 各有专属MDC
第三步：常规MDC分配
主诊ICD-10编码 → 决定进入哪个MDC大类（如F-循环系统、E-呼吸系统）
第四步：ADRG核心组确定
MDC + 主要手术/操作 → 确定ADRG（如FB1-心肌梗死手术组）
第五步：MCC/CC细分权重
其他诊断中是否有MCC（严重合并症）或CC（合并症）→ 决定最终DRG细分组和权重
主诊断"三最原则"
对患者危害最大
对本次住院影响最严重的疾病或状况，通常是导致住院的主要原因。
消耗医疗资源最多
诊治过程中消耗检查、操作、药物、耗材最多的疾病。
住院时间最长
为诊治哪个疾病住院时间最长，通常就是主诊断。
绝对禁止作主诊（无例外）
⛔ 症状编码（R系列）
发热R50.x、胸痛R07.x、呼吸困难R06.x、意识障碍R41.x等。有明确原发病时禁止作主诊。
⛔ 呼吸衰竭（J96.x）
是原发病的继发状态，必须列为MCC。主诊选重症肺炎/COPD/ARDS/脓毒症等原发病。
⛔ 附加编码
脓毒性休克R57.2、SIRS R65.1、心源性休克R57.0、DIC D65均为附加编码，绝不作主诊。
⛔ 1849条不入组编码
CHS-DRG 2.0明确规定不参与分组的编码，用于主诊会导致入组失败（Ungroupable）。
⛔ 医保灰码
以00结尾的非特指编码，下级有更精确分类时不应使用，会导致低权重或入组失败。
四川省特殊说明
执行时间
2025年1月1日起，四川省统一执行四川DRG分组目录（基于国家CHS-DRG 2.0），各市州采用点数法付费，基准点数和系数由各地确定。
特例单议（5%以内）
适用于：① 住院时间长、费用高的危重症；② 新药新耗材新技术病例；③ DRG现行分组未能涵盖的罕见病例；④ CAR-T等新兴治疗方式。申报比例不超过总病例5%。
⚠️ 高倍率病例
实际费用超过DRG基准费用约3倍时（各市阈值略有不同），医保按基准费用结算，差额由医院承担。应及时申请特例单议，避免医院亏损。
⚠️ 低倍率病例
实际费用低于DRG基准费用约50%时，医保按实际费用结算。应排查是否存在分解住院或提前出院，避免医保稽查。
"双通道"药品
双通道药品（部分生物制剂/CAR-T等）暂不纳入DRG付费，单独核算，不影响DRG权重结算。
⚠️ 合规底线：诊断排序须真实反映临床诊疗过程。通过虚增诊断、人为调整主诊高套权重属违规，医保稽核将追回款项并处罚。
手术操作对权重的影响（参考值）
内科保守治疗（无手术） RW 0.5 ~ 1.5
无手术操作编码，进入内科DRG组，权重最低。MCC/CC可提升至1.5左右。
一般手术（阑尾切除/腹腔镜胆囊等） RW 1.5 ~ 2.5
常见择期手术，进入外科DRG组，权重约为内科组2倍。合并MCC可升至3.0。
中等手术（关节置换/PCI单支/肿瘤根治等） RW 2.5 ~ 4.5
手术难度中等，技术要求高，是大多数科室的主要高权重分组区间。
复杂手术（多节段脊柱融合/多支血管PCI等） RW 4.5 ~ 8.0
技术难度大、耗材多、手术时间长，须有完整操作记录和内固定/耗材使用记录。
极复杂手术（TAVR/ECMO辅助/多脏器联合切除等） RW 8.0 ~ 15.0+
顶级术式，需多学科团队、高值耗材、完整评估记录，是医院DRG最高权重来源。
